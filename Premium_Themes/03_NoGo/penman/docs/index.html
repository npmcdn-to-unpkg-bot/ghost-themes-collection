  2. Restart Ghost instance.

Configuration

Twitter Cards

Set up your Twitter profile in partials/twitter-cards.hbs:

{{! Set up your Twitter profile }}
<meta name="twitter:site" content="@username">
<meta name="twitter:creator" content="@username">

Social links

You can set up links to your social profiles in partials/social-links.hbs:

<section class="social-links">
    {{! RSS Feed }}
    <a class="icon icon-feed" href="{{@blog.url}}/rss/" target="_blank">
        <i class="fa fa-rss"></i>
    </a>

    {{! Twitter }}
    <a class="icon icon-twitter" href="http://twitter.com/" target="_blank">
        <i class="fa fa-twitter"></i>
    </a>

    {{! Facebook }}
    <a class="icon icon-facebook" href="http://facebook.com/" target="_blank">
           <i class="fa fa-facebook"></i>
    </a>

    {{! Google Plus }}
    <a class="icon icon-google-plus" href="http://plus.google.com/" target="_blank">
        <i class="fa fa-google-plus"></i>
    </a>
</section>

You can add another social icons if you want because Penman uses Font Awesome Icons. You can check out all social icons here.

If you want to add a new icon, you need to change social-links.hbs in partials folder. Just add new link block:

<a class="icon icon-name" href="http://example.com/" target="_blank">
    <i class="fa fa-name"></i>
</a>

Now replace fa-name here <i class="fa fa-name fa-lg"> with icon code (fa-instagram, fa-linkedin, etc) from page that mentioned before.

Disqus comments

If you would like to use Disqus comments just change your shortname in partials/config.hbs:

{{! Disqus shortname }}
var disqus_shortname = 'example';

Google Analytics

If you would like to use Google Analytics just change your Google Analytics ID in partials/config.hbs:

{{! Google Analytics ID }}
var ga_id = 'UA-XXXXX-X';

Static pages menu

Ghost Aton (0.4) bring to us static pages feature. So if you want to add links of static pages in menu, you need to modify partials/navigation.hbs.
